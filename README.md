# CAQ Policy & Document Wiki — static preview

A self-contained snapshot of the proposed Croquet Queensland policy & document
wiki, for review. Static HTML — the live version is edited in the MyCroquet
backend (one source of truth) and rendered on the CAQ website.

- `index.html` — the wiki index (policies by category, roles, forms, minutes)
- one page per policy / role, plus `forms.html` and `minutes.html`
- content is a point-in-time copy; the real pages update from the CMS

Not the live site. No backend, no login, links to the CMS/downloads are
representative.
